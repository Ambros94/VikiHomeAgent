Word2VecDomainOperationFinder needs a trained word2vec model. Currently we are using the one provided by google.
Due to it's dimension (1.5Gb) it is not in the repository.
The model has to be located in the directory resources/word2vec/

obviously DO NOT RENAME THE DOWNLOADED FILE.
